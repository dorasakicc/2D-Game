2D Igrica (C# / Visual Studio)

Ovaj projekt predstavlja jednostavnu 2D igricu razvijenu u programskom jeziku C# korištenjem Visual Studija. Igrica je izrađena u sklopu fakultetskog projekta s ciljem primjene osnovnih koncepata objektno orijentiranog programiranja i rada s grafičkim elementima.

Projekt uključuje logiku igre, upravljanje objektima te osnovnu interakciju korisnika s igrom. Naglasak je stavljen na strukturu koda, jasnoću implementacije i razumijevanje OOP principa.

Korištene tehnologije:

-C#

-Visual Studio

-Objektno orijentirano programiranje (OOP)

Napomena

Projekt je edukativnog karaktera i razvijen kao dio studentskog rada.
