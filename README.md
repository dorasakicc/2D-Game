2D Game (C# / Visual Studio)

This project presents a simple 2D game developed in the C# programming language using Visual Studio. The game was created as part of a university assignment with the goal of applying fundamental object-oriented programming concepts and working with graphical elements.

The project includes core game logic, object management, and basic user interaction. The focus was placed on code structure, clarity of implementation, and understanding of OOP principles.

Technologies used:

-C#

-Visual Studio

-Object-Oriented Programming (OOP)

Note:
This project is educational in nature and was developed as part of student coursework.
